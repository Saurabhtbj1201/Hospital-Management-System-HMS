## вңЁ Overview

**Hospital Management System (HMS)** is a comprehensive, full-stack, open-source healthcare management platform designed to streamline hospital operations. It features **role-based access control** with dedicated portals for **Admins**, **Receptionists**, and **Doctors**, along with a **public-facing site** for patients to book appointments online.

<div align="center">

```
рҹҸҘ One Platform вҖ” Three Powerful Portals вҖ” Infinite Possibilities
```

| рҹҢҗ Public Site | рҹ”җ Admin Portal | рҹ‘©вҖҚвҡ•пёҸ Receptionist Portal | рҹ©ә Doctor Portal |
|:-:|:-:|:-:|:-:|
| Appointment Booking | Full System Control | Operations & Billing | Clinical Management |
| Department Browsing | User & Role Management | Patient Records | Appointment Handling |
| Doctor Discovery | Analytics Dashboard | Invoice Generation | Profile & Availability |


```bash

# git checkout -b docs/update-api-docs
```

**4пёҸвғЈ Make Your Changes**

- Write clean, commented code
- Test your changes locally

**5пёҸвғЈ Commit Your Changes**

```bash

```

> Use [Conventional Commits](https://www.conventionalcommits.org/) format:
> - `feat:` вҖ” New feature
> - `fix:` вҖ” Bug fix
> - `docs:` вҖ” Documentation changes
> - `style:` вҖ” Code formatting (no logic change)
> - `refactor:` вҖ” Code restructuring
> - `test:` вҖ” Adding/updating tests

**6пёҸвғЈ Push to Your Fork**

```bash
git push origin feature/your-feature-name
```

**7пёҸвғЈ Open a Pull Request & Submit Contribution Form**

- Go to the original repository and click **"New Pull Request"**
- Provide a clear title and description of your changes
- Link any related issues using `Fixes #issue-number`

> рҹ“қ **Final Step:** After opening your PR, please fill out the contribution form by clicking the **"Contribute Now"** button on the [HMS Contribution Page](https://projects.gu-saurabh.site/opensource/contribute-hospital-management-system-hms). This helps us track and acknowledge your contribution!
